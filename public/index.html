Datamodel Analysis Service

Pages

  • /visualize Shows an interactive diagram of all data structures in the triplestore.

Other data

  • /cache-warmed-up true if the essential queries have been cached
  • /all-graphs a list of graphs in the triplestore which will be included/excluded in the analysis (configure in config/excludedgraphs.json)
  • /all-relationships all (non-literal) relationships between resources in each non-excluded graph
  • /all-attributes all (literal) attributes for resources in each non-excluded graph
